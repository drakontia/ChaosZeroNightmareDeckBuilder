## V2からの変更点
2026年2月4日より適用される主要な変更点は以下の通り：

### 曖昧な記憶のポイント計算の変更

シーズン2では、セーブデータの保存規定が大幅に変更され、曖昧な記憶のポイント計算方法が以下のように調整されます。

#### 計算基準の変更

- 固有に獲得できる**開始カードとヒラメキカード、合計8枚**で構成された基本セーブデータを基準に、
   追加で獲得した要素を中心として価値が計算されます。
- 基本セーブデータの8枚には曖昧な記憶ポイントは加算されません。

#### 共用カードとモンスターカード

- **共用カード**はヒラメキの有無とは関係なく、**固定20pt**として計算されます（禁忌カードも同様）。
- **神ヒラメキ**は従来と同じく**20pt**で計算されます（基本カードは対象外）。
- **モンスターカード**は効果に応じて、**一般/希少/伝説グレード**に分けられ、
   それぞれの等級は**20pt/50pt/80pt**として計算されます。

#### カードのコピー

- カードのコピーは、**2回まではポイントとして計算されず、それ以降は1回につき40pt**として計算されます。
  - 1回目のコピー: **0pt**（コピー元の属性は継承）
  - 2回目のコピー: **0pt**
  - 3回目のコピー: **40pt**
  - 4回目のコピー: **40pt**（これが上限、5回目以降はコピー不可）

#### カードの排除（削除・変換）

- **開始カードを除くカードの排除**はポイントとして計算されません。
- **排除された開始カード**は1枚につき**20pt**として計算されます。
- **カード変換**も排除として計算されます（変換先カードには+10ptの基本変換コストが加算される従来ルールは廃止）。

#### 排除とコピーの上限

- 1つのセーブデータを基準として、以下の制限が適用されます：
  - **コピー**: 最大**4回**まで
  - **排除（削除・変換）**: 最大**5回**まで

#### 重要な変更点のまとめ

| 項目 | シーズン1 | シーズン2 |
|------|----------|----------|
| 基本カード（8枚） | ポイント加算あり | ポイント加算なし |
| 共用カード | 20pt（+ヒラメキ10pt） | 20pt（固定） |
| モンスターカード | 80pt（固定） | 一般20pt/希少50pt/伝説80pt |
| カードコピー | 0pt → 10pt → 30pt | 0pt → 0pt → 40pt |
| 開始カード以外の排除 | 回数に応じて加算 | 0pt（加算なし） |
| 開始カードの排除 | 回数に応じて加算 | 20pt（固定） |
| カード変換 | +10pt（基本コスト） | 排除として扱う（+10pt廃止） |
| コピー上限 | なし | 4回まで |
| 排除上限 | なし | 5回まで |

### トレサのリワーク

#### シーズン2バランス調整概要
苦痛システムの仕様変更に伴い、トレサのカード全体が調整されます。主な変更点は以下の通り：

- **苦痛システムの変更**: 苦痛が付与者基準で1つのデバフとして重複適用される方式に変更
- **苦痛の最大重複数**: 20重複に制限
- **トレサの戦闘位置付け**: 苦痛を軸とした立ち回りが更に強化される

#### 調整の背景
苦痛システムの動作方式が「付与者基準で1つのデバフとして重複適用」に変更されたことで、トレサの苦痛管理戦略がより直感的になります。従来の複数デバフ方式から単一デバフ方式へ移行することで、以下の改善が実現：

- 苦痛の重複数に応じたダメージ期待値がより明確化
- トレサのカード効果が苦痛数に依存する設計により、シナジーが強化
- 影の短剣のコスト効率改善により、序盤～中盤の展開力向上
- 上級影の短剣がコスト0になることで、苦痛スケーリングの活用機会拡大
